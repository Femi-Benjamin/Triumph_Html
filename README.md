# Triumph HTML

A simple landing page for Triumph Anointing — motion designer and video editor.

## Overview

This repository contains a single static HTML page:

- `index.html` — a polished hero section, work showcase, toolkit, about, and contact section.

## View locally

Open `index.html` directly in your browser, or run a local server from the project folder:

```powershell
cd "c:\Users\USER\Documents\NEXT\Triumph"
python -m http.server 8000
```

Then visit `http://127.0.0.1:8000/`.
